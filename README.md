Simple multiplayer online RPG. Mostly to practive python and learn a bit about networked game development.

```
cd client
./start.sh
cd server
./start.sh
```


FEATURES:
 - Create maps using Tiled map editor (http://www.mapeditor.org/)
 - Spawn monsters
 - Basic combat
 - Items, weapons, armor, shops
